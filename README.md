## Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar e consolidar meus estudos em React Native. Trata-se de um gerador de senhas seguras, que incluem letras maiúsculas e minúsculas, caracteres especiais e números.

O tema lúdico foi inspirado no desafio do curso, onde deveríamos criar um aplicativo para a Wayne Enterprise, a empresa do Batman.

## Como funciona?
Para gerar uma senha, basta clicar em "GENERATE". Uma senha aleatória e segura será gerada e exibida no campo acima. Para copiá-la, basta tocar no campo e a senha será automaticamente copiada para a área de transferência. Após a cópia, a senha será removida do campo automaticamente.
